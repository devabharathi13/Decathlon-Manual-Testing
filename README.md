# Decathlon India Website – Manual Testing Project

## About the Project

This is a Manual Testing project performed on the Decathlon India E-commerce website.

The goal of this project was to understand the Software Testing Life Cycle (STLC) and validate the website's functionality by creating and executing test cases.

**Website:** https://www.decathlon.in/

---

## Modules Tested

* Homepage
* Login
* Registration
* Search Product
* Product Details Page
* Add to Cart
* Cart
* Wishlist
* Checkout
* Order Confirmation
* Contact Us
